# Basic Discord Bot 

`🤖 Includes : ` ⬇

 - Kick command
 
 - Ban command
 
 - Animated Status
 
 - Member Add / Leave | Role create / add / remove | Message edit / delete LOGS
 
 - Server Members / Server Boosts status
 
 `🏹 Setup : ` ▶
 
 - Download **Basic Discord Bot**
 
 - Install Modules
 > npm install 
 > npm install discord.js
 > npm install glob
 > npm install util

 - Configure Bot (config.json)
 
 - Run Bot
 > node index.js
